= kotlin-springboot

== *springboot codebase*

.stack:
* webflux
* r2dbc h2
* rest api
* spring security jwt
* coroutines
* Http problem details RFC7807
* junit5
* gradle
* gradle-kotlin-dsl
* google-jib
* google-appengine
* mailSlurp-api
* gmail-api
* task scheduling
* cli: command line interface
* github-action(todo)

.features:
* Signup account
* Password reset and update
* Account management


==== **to test the app**
Open terminal and run : ```./gradlew check```

==== **to run the app**
Open terminal and run : ```./gradlew bootRun``` +
and launch http://localhost:8080/

==== **to run the cli**
Open terminal and run : ```./gradlew cli``` +

link:cadrage_webapp.adoc[Project phases in french]
