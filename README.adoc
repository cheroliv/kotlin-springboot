==== **to test the app**
Open terminal and run : ```./gradlew :monolith:check```

==== **to BDD test the app**
Open terminal and run : ```./gradlew :monolith:cucumber```

==== **to run the app**
Open terminal and run : ```./gradlew :monolith:bootRun``` +
and launch http://localhost:8080/
