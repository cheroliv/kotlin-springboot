= kotlin-springboot

== *springboot codebase*

.stack:
* springboot
* webflux
* spring data r2dbc
* rest api
* spring security jwt
* kotlin coroutines
* Http problem details

.features:
* Signup account
* Password reset and update
* Account management

==== **to test the app**
Open terminal and run : ```./gradlew check```

==== **to run the app**
Open terminal and run : ```./gradlew bootRun``` +
and launch http://localhost:8080/
